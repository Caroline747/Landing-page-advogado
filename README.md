# Landing page para advogado

Este e um repositório de escritório de advocacia, desenvolvido com foco em profissionalismo, clareza e credibilidade.  
O site apresenta informações sobre áreas de atuação, perfil profissional, formas de contato e conteúdos jurídicos, oferecendo uma experiência simples e acessível aos visitantes.

O objetivo é fortalecer a presença digital do advogado(a) e facilitar o contato com clientes de forma ética e informativa.

## Stack utilizada

**Front-end:** HTML, CSS, JavaScript e Bootstrap
